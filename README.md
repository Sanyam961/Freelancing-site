# SKILLIFY Freelance Marketplace

A premium, real-time freelance platform connecting visionary clients with top-tier talent.

- **Frontend**: Next.js 16 (React 19) styled with Tailwind CSS, utilizing Framer Motion and Rive for premium animations.
- **Backend & Database**: Convex serverless database for instant, real-time data synchronization.
- **Authentication**: Clerk integration for secure authentication and role-based user routing.
- **Payments**: Razorpay payment gateway integration for secure checkout and milestone completion.
